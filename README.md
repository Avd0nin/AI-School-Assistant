# 🤖 PhantomHelper AI

> Умный AI-помощник для учебы, подготовки к тестам и решения задач по программированию.

---

## 🚀 О проекте

**PhantomHelper** — это веб-приложение, которое помогает учиться быстрее и эффективнее с помощью искусственного интеллекта.

Он умеет:
- 📚 создавать подробные конспекты
- 🧠 объяснять сложные темы простым языком
- 📝 генерировать тесты и проверять ответы
- 💻 создавать контесты по программированию
- ⚡ запускать и проверять код (Python / C++)

---

## 🔥 Основные возможности

### 📖 Генерация конспектов
- подробные и структурированные объяснения
- HTML + поддержка LaTeX формул
- идеально для подготовки к контрольным

---

### 💬 AI-чат
- отвечает как умный наставник
- понимает контекст диалога
- умеет объяснять и помогать с кодом

---

### 📝 Генерация тестов
- 10 вопросов на тему
- варианты ответов + открытые вопросы
- автоматическая проверка через AI

---

### 🏆 Контесты по программированию
- генерация задач на лету
- разные уровни сложности
- примеры + тесты

---

### ⚔️ Проверка кода
Поддержка:
- 🐍 Python
- ⚙️ C++

Возможности:
- компиляция (для C++)
- запуск на тестах
- проверка вывода
- защита от зависаний (timeout)
- подробный результат по каждому тесту

---
